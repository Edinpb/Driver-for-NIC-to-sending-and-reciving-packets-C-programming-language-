1) Code is for Linux kernel.
2) Code is Compilng with MAKE file which is given in document. With MakeFile code is embed as a module in the Linux kernel
3) Make file is compiled in terminal with command: $ make -C /lib/modules/(uname -r)/build M=$PWD modules
4) $ is for sudo.

Transmiting part of this driver enables to send ranodom message to devices in LAN network.
Read method is for reading ping packets(actually payload of ping packets), this can be changed by simply moving the pointers.
